# CodeAlpha Task 1 — Web Scraping

## Overview
Scrapes book data from books.toscrape.com using Python, 
BeautifulSoup4 and Requests. Collects title, price, rating, 
category, stock status across all 50 pages.

## Files
- `task1_web_scraping.py` — main scraping script
- `books_scraped.csv` — scraped dataset
- `eda_report.txt` — analysis report

## Libraries Used
- requests
- beautifulsoup4  
- pandas

## How to Run
pip install requests beautifulsoup4 pandas
python task1_web_scraping.py
